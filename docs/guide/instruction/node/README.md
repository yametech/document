# 节点

::: tip

此版块包含集群中所有物理节点，目前共有6个节点，分为1主（master）和5从节点（slaver），可查看每个节点cpu、memory、disk的使用情况。点击每个节点，可查看该节点的详细信息，包含：cpu、memory、disk、pods等。

:::
